DIP SWITCH NO.2 SETTINGS

1.The Number of players lives
--------------------------------------------------
SW       1       2		      Number       
--------------------------------------------------
	off	off			1
0	on	off			2
	off	on			3
	on	on			5
--------------------------------------------------

2.The Difficulty of the Game
--------------------------------------------------
SW       6       7		   Difficulty       
--------------------------------------------------
	off	off		     Easy
0	on	off		    Normal
	off	on		   Difficult
	on	on		 Very Difficult
--------------------------------------------------

3.Sound in Attract Mode
--------------------------------------------------
SW       8		      	      Sound        
--------------------------------------------------
	off			       off
0	on			       on
--------------------------------------------------
Nb. SW3,4,5 are not used

1.VIDEO SCREEN FLIP

	on			   Upside Down
--------------------------------------------------

2.CHANGE OF MODE
--------------------------------------------------
SW       1       		       Mode       
--------------------------------------------------
0	off			       Game	
	on			       Test
--------------------------------------------------
Nb. SW2,4 are not used
